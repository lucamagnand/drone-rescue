## Objectif
À la fin de ce module, l'étudiant sait coder `phase_j1(etat)` : lire une saisie clavier, la découper (parser), appeler `valider_mouvement()` puis `executer_mouvement()`, et boucler jusqu'à 3 déplacements ou que J1 passe son tour.

## Le QUOI — Définition et concept

La **console** est la couche qui fait le lien entre l'utilisateur et la logique du jeu. Elle ne contient aucune règle métier — elle lit, valide le format, délègue.

L'architecture de `phase_j1()` suit toujours le même pipeline :

```
 input()  →  parser  →  valider_mouvement()  →  executer_mouvement()
 (lire)     (découper)   (règles métier)        (appliquer)
```

La saisie attendue de J1 est de la forme `"D1 B4"` ou `"D1 passe"` :
- `"D1 B4"` : déplacer le drone D1 vers la colonne B, ligne 4
- `"D1 passe"` : J1 passe son tour (fin de sa phase)
- `"fin"` : mettre fin au tour de J1 immédiatement

```python
# Conversion de la notation humaine vers indices
# "B4" → col = COLS.index('B') = 1, lig = 4 - 1 = 3
COLS = 'ABCDEFGHIJKL'
col = COLS.index(chaine[0].upper())   # lettre → index 0-11
lig = int(chaine[1:]) - 1             # numéro (1-12) → index (0-11)
```

## Le POURQUOI — Enjeu pédagogique

Séparer le **parser** (découpage de la chaîne) de la **validation métier** (règles du jeu) est un pattern fondamental. Si J1 tape `"D1 ZZ"`, le parser détecte un format invalide avant même d'appeler `valider_mouvement()`. Si J1 tape `"D1 B4"` sur un bâtiment, c'est `valider_mouvement()` qui le rejette.

Ce découpage rend chaque couche testable indépendamment.

## Les RISQUES

### Si on ne le fait pas
Sans `phase_j1()`, J1 ne peut pas jouer. La boucle de jeu est un squelette sans interaction.

### Si on le fait mal
**Bug classique 1 — oublier la limite de déplacements.** Si le compteur `nb_depl` n'est pas vérifié, J1 peut déplacer ses drones indéfiniment.

```python
# ERREUR : pas de limite
while True:
    saisie = input("J1 > ")
    executer_mouvement(...)  # J1 joue à l'infini

# CORRECT : compteur sur MAX_DEPLACEMENTS_J1
nb_depl = 0
while nb_depl < MAX_DEPLACEMENTS_J1:
    saisie = input("J1 > ")
    ...
    nb_depl += 1
```

**Bug classique 2 — ne pas gérer les saisies mal formées.** Si l'étudiant ne met pas `try/except` ou de vérification de longueur, une saisie vide ou `"D1"` seul fait crasher `int(chaine[1:]) - 1`.

## Lien avec jeu/

Extrait commenté de `jeu/console.py` :

```python
COLS = 'ABCDEFGHIJKL'
MAX_DEPLACEMENTS_J1 = 3

def position_depuis_chaine(chaine):
    """
    Convertit une chaîne de position (ex : 'B4') en (col, lig).
    Retourne (col, lig) ou (None, None) si format invalide.
    """
    try:
        col = COLS.index(chaine[0].upper())
        lig = int(chaine[1:]) - 1
        if 0 <= col < 12 and 0 <= lig < 12:
            return col, lig
    except (ValueError, IndexError):
        pass
    return None, None


def phase_j1(etat):
    """
    Tour de J1 : jusqu'à MAX_DEPLACEMENTS_J1 déplacements de drones.
    Saisie : 'D1 B4' (déplacer D1 vers B4) ou 'fin'/'passe' pour terminer.
    """
    nb_depl = 0
    print(f"\n=== Tour J1 (max {MAX_DEPLACEMENTS_J1} déplacements) ===")

    while nb_depl < MAX_DEPLACEMENTS_J1:
        saisie = input(f"  J1 [{nb_depl}/{MAX_DEPLACEMENTS_J1}] > ").strip()

        if saisie.lower() in ("fin", "passe", ""):
            break

        parties = saisie.split()
        if len(parties) != 2:
            print("  Format attendu : D1 B4 (drone + position)")
            continue

        id_drone, pos_str = parties[0].upper(), parties[1]

        if id_drone not in etat["drones"]:
            print(f"  Drone {id_drone} inconnu.")
            continue

        if pos_str.lower() in ("passe", "fin"):
            break

        col, lig = position_depuis_chaine(pos_str)
        if col is None:
            print(f"  Position '{pos_str}' invalide. Exemple : B4")
            continue

        ok, msg = valider_mouvement(etat, id_drone, col, lig)
        if not ok:
            print(f"  Mouvement refusé : {msg}")
            continue

        log = executer_mouvement(etat, id_drone, col, lig)
        print(f"  ✓ {log}")
        etat["historique"].append(log)
        nb_depl += 1

    print(f"  J1 terminé ({nb_depl} déplacement(s))")
```
