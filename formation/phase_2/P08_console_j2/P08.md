## Objectif
À la fin de ce module, l'étudiant sait coder `phase_j2(etat)` (déplacement des tempêtes par J2) et `boucle_de_jeu(etat)` qui orchestre un tour complet : J1 → J2 → phase automatique → vérification de fin.

## Le QUOI — Définition et concept

Un **tour de jeu complet** se décompose en 4 phases distinctes, toutes orchestrées par `boucle_de_jeu()` :

```
Tour N
  1. phase_j1(etat)          ← J1 déplace ses drones (max 3)
  2. phase_j2(etat)          ← J2 déplace ses tempêtes (max 2)
  3. fin_de_tour(etat)       ← météo automatique + propagation X + tour++
  4. verifier_fin_partie()   ← victoire ? défaite ?
Tour N+1
  ...
```

Chaque phase est une fonction séparée. `boucle_de_jeu()` ne contient aucune logique métier — elle orchestre.

La saisie J2 suit le même pattern que J1, mais porte sur les tempêtes :
- `"T1 F6"` : déplacer la tempête T1 vers F6
- `"fin"` : J2 passe

La validation J2 est plus simple (pas de batterie) : vérifier que la case cible est dans la grille et à distance Chebyshev ≤ 1 de la position actuelle.

## Le POURQUOI — Enjeu pédagogique

`boucle_de_jeu()` illustre le concept d'**orchestrateur** : une fonction de haut niveau qui appelle des fonctions de bas niveau dans l'ordre correct. Sa valeur tient à sa simplicité : si on lit `boucle_de_jeu()`, on comprend immédiatement la structure du jeu, sans noyer le lecteur dans les détails.

C'est le principe de **séparation des niveaux d'abstraction** : `boucle_de_jeu()` pense en termes de « phases », pas en termes de « cases de grille ».

## Les RISQUES

### Si on ne le fait pas
Sans `boucle_de_jeu()`, le jeu ne peut pas se dérouler. On peut avoir tout le code des modules précédents ; sans l'orchestrateur, les pièces ne sont pas assemblées.

### Si on le fait mal
**Bug classique 1 — vérification de fin après incrémentation du tour.** Si on incrémente `etat['tour']` AVANT de vérifier `verifier_fin_partie()`, le message de fin peut afficher « Tour 51 » au lieu de « Tour 50 ».

```python
# ERREUR : tour incrémenté avant vérification
etat["tour"] += 1
msg = verifier_fin_partie(etat, TOURS_MAX=50)  # affiche tour 51

# CORRECT : vérifier d'abord
msg = verifier_fin_partie(etat, TOURS_MAX=50)  # affiche tour 50
etat["tour"] += 1
```

**Bug classique 2 — ne pas tester `etat['partie_finie']` en début de boucle.** Si la partie est déjà finie (par exemple après une défaite lors de la phase J1), la boucle `while` doit s'arrêter immédiatement au début du tour suivant.

## Lien avec jeu/

### `phase_j2()` dans `jeu/console.py`
```python
MAX_DEPLACEMENTS_J2 = 2

def valider_mouvement_tempete(etat, id_tempete, col_cible, lig_cible):
    """Vérifie que le mouvement de tempête est légal."""
    t = etat["tempetes"].get(id_tempete)
    if t is None:
        return False, f"{id_tempete} inconnue"
    # Distance Chebyshev ≤ 1
    if max(abs(col_cible - t["col"]), abs(lig_cible - t["lig"])) > 1:
        return False, "distance > 1 case"
    if not (0 <= col_cible < 12 and 0 <= lig_cible < 12):
        return False, "hors grille"
    return True, "ok"


def phase_j2(etat):
    """Tour de J2 : jusqu'à MAX_DEPLACEMENTS_J2 déplacements de tempêtes."""
    nb_depl = 0
    print(f"\n=== Tour J2 (max {MAX_DEPLACEMENTS_J2} déplacements) ===")

    while nb_depl < MAX_DEPLACEMENTS_J2:
        try:
            saisie = input(f"  J2 [{nb_depl}/{MAX_DEPLACEMENTS_J2}] > ").strip()
        except (EOFError, KeyboardInterrupt):
            break

        if saisie.lower() in ("fin", "passe", ""):
            break

        parties = saisie.split()
        if len(parties) != 2:
            print("  Format : T1 F6  |  'fin' pour terminer")
            continue

        id_tempete, pos_str = parties[0].upper(), parties[1]

        if id_tempete not in etat["tempetes"]:
            print(f"  Tempête '{id_tempete}' inconnue.")
            continue

        col, lig = position_depuis_chaine(pos_str)
        if col is None:
            print(f"  Position '{pos_str}' invalide.")
            continue

        ok, msg = valider_mouvement_tempete(etat, id_tempete, col, lig)
        if not ok:
            print(f"  ✗ Refusé : {msg}")
            continue

        # Exécution : déplacer la tempête
        t = etat["tempetes"][id_tempete]
        etat["grille"][t["lig"]][t["col"]] = "."
        t["col"], t["lig"] = col, lig
        etat["grille"][lig][col] = id_tempete
        log = f"{id_tempete} → ({col},{lig})"
        print(f"  ✓ {log}")
        etat["historique"].append(f"J2 | {log}")
        nb_depl += 1

    print(f"  → J2 : {nb_depl} déplacement(s) effectué(s)")
```

### `boucle_de_jeu()` dans `jeu/console.py`
```python
def boucle_de_jeu(etat, TOURS_MAX=50):
    """Orchestre le jeu : J1 → J2 → météo → fin de partie, tour après tour."""
    from .affichage import render_grille
    from .logique import (executer_mouvement, valider_mouvement,
                          deplacer_tempetes, propager_zones_x,
                          verifier_fin_partie)

    while not etat["partie_finie"]:
        render_grille(etat)
        print(f"\n─── Tour {etat['tour']} | Score : {etat['score']} ───")

        # 1. Phase J1
        phase_j1(etat, valider_mouvement, executer_mouvement)
        if etat["partie_finie"]:
            break

        # 2. Phase J2
        phase_j2(etat)
        if etat["partie_finie"]:
            break

        # 3. Phase automatique
        deplacer_tempetes(etat)
        if etat["tour"] % 5 == 0 and etat["zones_x"]:
            propager_zones_x(etat)

        # 4. Vérification fin + incrémentation tour
        msg_fin = verifier_fin_partie(etat, TOURS_MAX)
        etat["tour"] += 1

        if msg_fin:
            render_grille(etat)
            print(f"\n{'='*40}")
            print(msg_fin)
            print(f"Score final : {etat['score']}")
            break
```
