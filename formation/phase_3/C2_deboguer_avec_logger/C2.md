## Objectif
À la fin de ce module, l'étudiant sait lire une stack trace Python, utiliser `print()` de débogage, lire un fichier de log, et identifier 3 bugs intentionnels dans un code fourni grâce aux traces produites.

## Le QUOI — Définition et concept

Un **bug** = un écart entre l'état attendu et l'état réel du programme. Déboguer, c'est **réduire cet écart** en rendant l'état visible.

Deux outils principaux :

1. **`print()` de débogage** : rapide, directement dans le code, à supprimer après.
2. **Logger** : écrit dans un fichier texte ; persistant, lisible même après une erreur.

Le logger de Drone Rescue écrit chaque action du jeu dans `partie_AAAA-MM-JJ.log` :
```
[Tour 3] D1 se déplace de B4 vers C4. Batterie : 7
[Tour 3] D1 embarque S2 en C4.
[Tour 3] T1 se déplace automatiquement vers D6.
```

Ces traces permettent de rejouer mentalement la partie et de vérifier que chaque action est correcte.

## Le POURQUOI — Enjeu pédagogique

Sans trace, un bug dans la phase automatique (déplacement des tempêtes, propagation des zones X) est presque impossible à reproduire. Avec le logger, on sait exactement quel tour, quelle action, quel drone, quelle tempête était impliquée.

## Les RISQUES

### Si on ne le fait pas
Le débogage se fait "au flair" : on modifie le code sans comprendre pourquoi ça ne marchait pas, et on introduit de nouveaux bugs en corrigeant l'ancien.

### Si on le fait mal
```python
# MAUVAIS : print() laissés dans le code final
def executer_mouvement(etat, id_drone, col, lig):
    print("DEBUG executer_mouvement appelé")
    print(f"DEBUG drone = {etat['drones'][id_drone]}")
    # ... logique ...
```

Un `print()` de débogage laissé en production pollue la sortie du jeu et peut révéler des informations à J2 (position des survivants, état des drones). Utiliser le logger à la place : les traces vont dans un fichier, jamais dans le terminal de jeu.

## Lien avec jeu/

Extrait de `jeu/logger.py` :

```python
import datetime
import os

_fichier_log = None

def demarrer_log(dossier='.'):
    """Crée un nouveau fichier de log pour la partie."""
    global _fichier_log
    nom = f"partie_{datetime.date.today()}.log"
    _fichier_log = open(os.path.join(dossier, nom), 'a', encoding='utf-8')
    ecrire_log("=== Nouvelle partie ===")

def ecrire_log(message):
    """Ajoute une ligne dans le fichier de log."""
    if _fichier_log:
        _fichier_log.write(message + '\n')
        _fichier_log.flush()

def fermer_log():
    """Ferme le fichier de log proprement."""
    global _fichier_log
    if _fichier_log:
        _fichier_log.close()
        _fichier_log = None
```
